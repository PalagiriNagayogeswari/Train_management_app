# 🚆 Train Consist Management System

A Java-based project developed as part of the Bridgelabz Java Training Program.

This project demonstrates Java fundamentals, Collections Framework, Streams API, Exception Handling, Searching, Sorting, Regular Expressions, and Algorithms through a real-world **Train Consist Management System**.

---

## 📚 Technologies Used

- Java
- Collections Framework
- Streams API
- Exception Handling
- Regular Expressions
- Arrays
- Searching Algorithms
- Sorting Algorithms
